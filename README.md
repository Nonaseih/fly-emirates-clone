# Emirates Clone (Navbar Prototype)

Quick React prototype using Vite and Tailwind via CDN.

Getting started:

1. Install deps

```powershell
npm install
```

2. Run dev server

```powershell
npm run dev
```

Open the provided Vite URL (usually http://localhost:5173).

Fonts:
- To use the `EK0plain - M02` logo font, place the font file (e.g. `EK0plain-M02.woff2`) under `public/fonts/` so it's available at `/fonts/EK0plain-M02.woff2`.
- Nav links use the system UI sans stack by default; replace or add webfonts in `src/styles.css` if desired.
